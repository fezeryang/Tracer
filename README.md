# NUX AI Financial Research Terminal

NUX 智能金融研究终端 contains everything you need to run the app locally.

View your app in AI Studio: https://ai.studio/apps/29da9528-308d-4482-9772-c6a28a7f0899

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
